# Revo 数据结构

## 一、现有 Mock 数据文件

| 文件 | 内容 |
|---|---|
| `data/homeData.js` | 智脑首页：今日市场摘要、机会 TOP3、趋势、AI提醒 |
| `data/inventoryData.js` | 库存大脑：总览、诊断、健康度、经营行动、趋势、导入字段 |
| `data/priceData.js` | 价格库 |
| `data/categoryData.js` | 品类数据 |
| `data/industryData.js` | 行业分析数据 |
| `data/marketHeat.js` | 收货/卖货/机会 热度 Treemap 数据 |

## 二、核心实体

### 商品 SKU

<!-- brand / model / capacity / condition / battery -->

### 库存条目

<!-- 品牌 / 型号 / 容量 / 成色 / 采购价 / 数量 / 采购日期 -->

### 机会条目

<!-- name / score / reason[] / suggest / price -->
